My Work

Below are some of my recent projects in UX, XR, and frontend development. Each project showcases my dedication to constantly learning and evolving as a designer and developer. Feel free to explore each project in detail to see the tools and techniques I used, along with the unique challenges I faced and how I overcame them.

Weather App

This weather app was built as part of my SheCodes project, using JavaScript and the OpenWeather API to provide real-time weather data and forecasts. The app allows users to search for any city and view current weather, temperature, humidity, and a forecast for the next days. This project helped me improve my skills in working with APIs and responsive design.
